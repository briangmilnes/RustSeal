## Columns

- **#** — continuous row id (rule 15.1).
- **directory** — an immediate subdirectory of the scan root.
- **files** — distinct `.rs` files parsed (`target/` skipped).
- **forget / forget_unsized / manually_drop / into_raw / leak** — occurrences per leaf category.
- **total** — `forget + forget_unsized + manually_drop + into_raw + leak`.
- **unsafe_fn** — of `total`, occurrences inside an `unsafe fn`.
- **not_unsafe_fn** — of `total`, occurrences NOT inside an `unsafe fn` (safe-fn bodies, `unsafe { }` blocks within a safe fn, and module scope). `total = unsafe_fn + not_unsafe_fn`.
- **in_unsafe_block** — of `not_unsafe_fn`, occurrences lexically inside an `unsafe { }` block (safe signature, unsafe body).
- **mod_has_unsafe_fn** — of `not_unsafe_fn`, occurrences whose enclosing module also declares an `unsafe fn`.
- `—` = not-applicable; `unknown` = existed-but-unmeasured; `0` = measured zero (kept distinct).
